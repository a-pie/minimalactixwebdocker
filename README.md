## How to use

```text
This is a minimalist actix web 'hello world' example that runs inside of a docker container. The dockerfile is setup to massively reduce both image size and compile times via use of cargo-chef and a mini debian image.
See commands.txt for the basics
See https://actix.rs/ and https://www.youtube.com/watch?v=xuqolj01D7M for full tutorials.
```
